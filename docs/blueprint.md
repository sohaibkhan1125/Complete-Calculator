# **App Name**: CalcComplete

## Core Features:

- Scientific Calculator: Display a fully functional scientific calculator on the left side of the main section.
- Calculator Search: Implement a search input field and button for users to search calculators by name.
- Calculator Categories: Show four calculator categories: Financial, Fitness & Health, Math, and Other, with rounded icons/images and category titles. Provide clickable placeholders for calculator cards within each category.
- Footer: Place a full-width footer at the bottom of the page with copyright information, and links for About, Contact, and Privacy Policy.
- AI-Powered Calculation: Allow users to describe calculations they want to perform in natural language, and use an LLM to interpret their requests, use a scientific calculator tool, and display results.

## Style Guidelines:

- Primary color: HSL 220, 50%, 50% which translates to a vibrant blue (#337AB7) for the calculator and main elements.
- Background color: HSL 220, 20%, 95% which translates to a light grayish-blue (#F0F4F8) providing a soft and clean backdrop.
- Accent color: HSL 190, 60%, 50% which translates to a contrasting teal (#30A5A5) used for interactive elements and highlights.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable interface.
- Use simple and clear icons for calculator categories and search button.
- Responsive design using a CSS utility framework (Tailwind CSS) to ensure compatibility across devices (desktop, tablet, mobile).
- Subtle hover effects and transition animations for buttons and interactive elements to enhance user engagement.
